# 🖥️ Это минималистичное и быстрое веб-приложение для сбора прокси-конфигураций из различных источников.

![Status](https://img.shields.io/badge/Status-Active-success?style=flat-square)
![Platform](https://img.shields.io/badge/Platform-Web-white?style=flat-square)
![License](https://img.shields.io/badge/License-MIT-black?style=flat-square)

---

## ⚡️ Особенности
* **Auto-Cleaning:** Автоматическое удаление дубликатов и лишнего текста из списков.
* **Static Powered:** Работает полностью на клиенте (GitHub Pages) без бэкенда.

## 🛠️ Поддерживаемые протоколы

Программа фильтрует и собирает следующие типы ссылок:
- `VLESS://`
- `VMess://`
- `Trojan://`
- `Shadowsocks (SS)://`

## 💻 Стек технологий

* **HTML5** — структура.
